Fecha: 19/01/2022 Categoría: Javascript

Finalización del proyecto

En la última semana programamos los diferentes ficheros de js. Comenzamos con la integración de un carrusel de imágenes (visual lightbox) gracias a JQUERY, donde no solo incluyo la librería sino que debo localizar dentro de la misma las diferentes funciones para adaptarla a mi proyecto. Posteriormente continuamos con la validación del formulario de la sección "Presupuesto". El descuento lo he calculado descontando un 10% del total según las semanas que nos cedan para crearlo (obviamente, esto es irreal en la vida real, simplemente es un ejemplo). Por último integramos la API de Google Maps para insertar en la sección "Contacto" el mapa con localización de la empresa, con una marca que nos muestra dó está y có se llama y un calculador de ruta que nos muestra la ruta desde la ubicación que le ingresemos hasta la empresa.

PD: El buscador del index no está funcionando porque no he querido emplear más tiempo para integrar una función que no es necesaria para el proyecto. Símplemente lo he diseñado para crear un aspecto visual atractivo.

Fecha: 12/01/2012 Categoría: JSON

Incluímos el lector de JSON

Incluimos un lector de json gracias a JavaScript. El documento .json lo inluyo en una carpeta llamada "resources" que espero me sirva má adelante para organizar otros archivos que pueda ir necesitando a la hora de crear una aplicación web.

Me he valído del panel "aside" derecho para incrustar las diferentes noticias de temática relacionada con el curso. Esto proporcionará un aspecto más claro y elegante, permitiendo al usuario ignorar las noticias (en caso de no estar interesado en ellas) o por el contrario, tener un rápido acceso a las mismas.

Fecha: 06/01/2022 Categoría: Información

Iniciamos la plataforma de PAX | DIGITAL MINDS

Comenzamos la aventura de PAX | DIGITAL MINDS creando el esqueleto del HTML. Basándonos en el anterior proyecto (Max Serratosa | Full stack developer), tomamos algo de la estructura (como por ejemplo el encabezado que incluye un menú de navegación o el footer con el script que utilizamos también para el "hover" de las redes sociales).

Incrustamos el CSS incorporando conocimientos que he ido añadiendo durante estos meses gracias a cursos externos, información de diferentes webs o perfiles de instagram que enseñan fórmulas de JavaScript y css. Esto me permite dar un aspecto mucho más atractivo a la web con diferentes efectos que ofrecen dinamismo visual. Además, he incrustado diferentes tipos de fuentes que dan un aspecto profesional al trabajo final.
